The Cold Case Files

Three real experiments from the 2000s changed neuroscience forever. The electrode recordings from those rat sessions are bundled into this codebase. Here is how the code finds, downloads, and uses that data — and why it matters for model validation.

Think of the Experiment classes as cold case detective files. Scientists wired real rats with electrodes, recorded exactly which neuron fired at which location, and those recordings became a permanent historical record. NeuralPlayground downloads, caches, and provides access to those files.

Experiments provide two things: real behavioral data trajectories (so your model follows the same path a real rat took) and real neural spike train data (so you can compare firing patterns). This comparison IS the scientific contribution.

Replaying a Real Experiment

When you pass use_behavioral_data=True, the arena stops being a generic box and becomes a precise replay of the original lab session. Here is what each line does — from the experimental arena usage pattern.

CODE 
from neuralplayground.agents import Stachenfeld2018
from neuralplayground.arenas import Sargolini2006

env = Sargolini2006(use_behavioral_data=True)
agent = Stachenfeld2018()

obs, state = env.reset()
for step in range(env.total_number_of_steps):
    action = agent.act(obs)
    obs, state, reward = env.step(action)
    agent.update()
PLAIN ENGLISH

import the successor representation brain model

import the arena that wraps the Sargolini experiment

Load a virtual replica of the Sargolini lab experiment, AND load the actual rat trajectory from disk (auto-downloads if not cached)

Create the brain model — this is the theory we want to test against real data

Place the agent at the same starting position a real rat used in the original experiment

Loop exactly as many steps as the real rat took in the experiment

Ask the model where to go next (or follow the real rat path if behavioral data overrides)

Move through the arena — potentially following the real rat trajectory

Let the model learn from this position, just as it would in any other arena

The key difference from a plain arena: env.total_number_of_steps is not a setting you choose — it is the exact length of the original rat recording. The loop ends when the replay ends.

Three Experiments, Three Stories

Each dataset captures a different aspect of spatial cognition. Together they cover the core cell types — grid cells, place cells, head direction cells — that any spatial navigation model needs to explain.

Hafting2008 Grid cell recordings from rat entorhinal cortex. The paper that discovered grid cells.
Sargolini2006 Place cell and head direction cell recordings. Multiple open-field sessions.
Wernle2018 Dynamic room merging paradigm — two rooms become one, cells remap.

Each dataset class exposes the same interface: a position array, a head_direction array, and a recording_list dataframe listing all available sessions. This uniformity means you can swap datasets without changing your analysis code.

To validate your model: run the simulation with use_behavioral_data=True, compute a rate map from the model's activations, then compare its spatial autocorrelation against the real cell's rate map. A high match means your model is doing something biologically plausible.
